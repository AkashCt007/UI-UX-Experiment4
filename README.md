# UI-UX-Experiment4

## Aim:

To design a simple low-fidelity wireframe for the  application using Balsamiq, focusing on the layout structure, navigation flow, and placement of UI elements across essential screens such as Splash Screen, Login/Signup, Home Dashboard, Music Listing, and Checkout Screen without using colors or detailed graphics.

## Algorithm:

1. Start the wireframe design process.
2. Create the Welcome/Splash Screen with logo and Get Started button.
3. Design the Login/Signup Screen with input fields and authentication buttons.
4. Develop the Home/Dashboard Screen with search bar, playlists, and navigation menu.
5. Create the Content/Music Listing Screen showing songs, albums, and playback options.
6. Design the Final Action/Checkout Screen with subscription plans and payment options.
7. Connect all screens using navigation arrows to represent user flow.
8. Arrange UI components clearly using simple grayscale placeholders.
9. Review the wireframe for proper layout structure and usability.
10. End the design process.
